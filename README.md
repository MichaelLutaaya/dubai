# Dubai
**Generate and Preview Passbook Passes**

## Installation

    $ gem install dubai


## Comand Line Interface

Dubai also comes with the `pk` binary, which provides a convenient way to generate and preview passes


    $ pk generate mypass
    $ pk serve mypass
    $ open http://localhost:4000
